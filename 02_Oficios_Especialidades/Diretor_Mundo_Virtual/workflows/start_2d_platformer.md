# Start 2D Platformer Project

This workflow scaffolds a new Godot 4 project complete with a CharacterBody2D player controller, physics settings, and a basic level template.

1. Run the scaffolding script (Replace `[project_name]` with your desired name)
```bash
python .agent/scripts/scaffold_2d_platformer.py [project_name]
```

2. Update the Documentation
