# Skill: Padrão de Entrega Melanora (L3)

Define o nível de excelência exigido para qualquer exportação ou release.

## 📝 Protocolos
1.  **Checklist de Artefatos**: Garantir que o `README.md` e o `walkthrough.md` do projeto estejam atualizados.
2.  **Limpeza de Debug**: Desativar `print` statements e visuais de colisão para a versão final.
3.  **Sanity Check de Exportação**: Rodar o jogo uma última vez no build exportado para garantir que nada quebrou no empacotamento.
