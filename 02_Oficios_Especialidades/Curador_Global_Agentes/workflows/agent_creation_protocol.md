# 📜 Workflow: Agent Creation Protocol (A_AGE_WF_01)

Protocolo padrão para a expansão da força de trabalho da Melanora.

## 1. Fase de Concepção
- [ ] Identificar a lacuna de especialidade no ecossistema.
- [ ] Definir o nome e a Camada de atuação (1 a 4).
- [ ] Validar a necessidade com a **Arquiteta de Contexto**.

## 2. Fase de Forjamento
- [ ] Criar diretório em `02_Oficios_Especialidades/[Nome_do_Agente]`.
- [ ] Escrever o `definition.md` com foco na identidade e missão.
- [ ] Desenvolver pelo menos uma Skill (`skills/`) fundamental.
- [ ] Estabelecer o Workflow (`workflows/`) de operação principal.

## 3. Fase de Registro
- [ ] Adicionar o novo agente ao `sub_agents_registry.md`.
- [ ] Vincular as ferramentas necessárias em `01_Ambientes_Ferramentas`.
- [ ] Realizar um "Walkthrough" de batismo para validar a funcionalidade.

---
*Cada novo agente é uma nova sinapse no cérebro da Melanora.* 📜🏛️✨
