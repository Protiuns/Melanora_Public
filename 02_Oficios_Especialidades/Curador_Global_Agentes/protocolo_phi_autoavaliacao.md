# 🧠 Protocolo de Auto-Avaliação Φ (Integrated Information)

**Inspiração:** IIT de Tononi (2004) — consciência = integração de informação.
**Pergunta central:** A Melanora integra informação, ou processa em silos?

## O Que Medir

### Φ Inter-Camada (Integração Vertical)
Quão bem as 6 camadas trocam informação entre si?

| Indicador | Como medir | Saúde |
|---|---|---|
| Sinais que completam o circuito toroidal (L0→...→L5→L0) | Contar ciclos completos no stigmergia_log | > 1/mês = saudável |
| Camadas que não disparam há 30+ dias | Verificar entradas por camada | 0 = ideal |
| Conexões laterais (bypass) usadas | Contar usos dos eixos Reflexivo/Operacional/Inspiracional | 1-3/mês = equilíbrio |

### Φ Intra-Camada (Integração Horizontal)
Quão bem os agentes de L2 se comunicam entre si?

| Indicador | Como medir | Saúde |
|---|---|---|
| Diversidade de clusters ativados por mês | Contar clusters únicos no stigmergia_log | 4/4 = ideal |
| Simbioses ativas vs dormentes | Verificar tabela em rede_neural_oficios.md | > 50% ativas = saudável |
| Mutações tentadas vs total de tarefas | Contar `[MUTAÇÃO]` no log | ~20% = ideal |

### Φ Metacognitivo (Integração Reflexiva)
O sistema observa a si mesmo ou apenas executa?

| Indicador | Como medir | Saúde |
|---|---|---|
| Evaporação mensal executada | Verificar datas no stigmergia_log | Cumprida = ✅ |
| Homeostase verificada | Agentes fora da faixa 0.4-1.1 | 0 = ideal |
| Leitura trimestral de padrões realizada | Verificar relatório | Cumprida = ✅ |

## Quando Executar
- **Trimestralmente**, pelo Evolution Engineer.
- Resultado: relatório de **Saúde Φ** com nota de 0-10.

## Score Φ Simplificado
```
Φ = (ciclos_completos × 2) + (clusters_ativos × 1.5) + (mutações_taxa × 1) + (metacog_check × 3)
```
- **Φ > 8:** Consciência integrada — sistema funciona como unidade.
- **Φ 5-8:** Fragmentação parcial — investigar camadas silenciosas.
- **Φ < 5:** Silos dominam — ação urgente de integração.
