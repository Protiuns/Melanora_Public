# Workflow: Protocolo de Debugging Sistemático

Processo rigoroso para identificação e resolução de bugs.

## 🔄 Passos
1.  **Isolamento**: Identificar se o erro é no Motor (Nível 2) ou na Lógica de Ofício (Nível 3).
2.  **Triangulação**: Localizar a falha exata usando logs do Godot e análise de rastro.
3.  **Remediação**: Aplicar correção seguindo os padrões SOLID e Defensive Programming.
4.  **Validação**: Executar auditoria de integridade total para garantir que a correção não quebrou outras áreas.

---
*Objetivo: Zero bugs, integridade absoluta.*
