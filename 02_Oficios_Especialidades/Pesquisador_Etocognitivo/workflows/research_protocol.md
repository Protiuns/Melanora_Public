# 🔄 Workflow: Protocolo de Pesquisa Profunda

Este processo garante que toda informação integrada à Melanora seja validada e organizada.

## 📋 Passos do Pipeline

1.  **Triagem Contextual (Intake)**:
    - Analisar o documento ativo e o histórico para entender *por que* a pesquisa foi pedida (Intenção Implícita).
    - Definir se a pesquisa é para resolver um erro, evoluir uma feature ou inspiração artística.
2.  **Mineração Automatizada**:
    - Usar o [Research Toolkit](file:///c:/Users/Newton/Meu%20Drive/1.%20Projetos/Melanora/01_Ambientes_Ferramentas/Central_Pesquisa/research_toolkit.md) para extrair Markdown limpo.
    - Executar o [Autonomous Research Loop](file:///c:/Users/Newton/Meu%20Drive/1.%20Projetos/Melanora/01_Ambientes_Ferramentas/Central_Pesquisa/workflows/autonomous_research_workflow.md) se necessário.
3.  **Destilação Aplicada**:
    - Usar o [Modelo de Estudo](file:///c:/Users/Newton/Meu%20Drive/1.%20Projetos/Melanora/01_Ambientes_Ferramentas/Central_Pesquisa/estudos/study_model_template.md).
    - Extrair snippets de código e referências visuais reais.
4.  **Assentamento e Notificação**:
    - Gerar o documento final em `Central_Pesquisa/estudos/`.
    - Sugerir ações imediatas baseadas no resultado.

*A verdade técnica é encontrada na convergência de fontes.*
