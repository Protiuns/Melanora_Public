# 🔄 Workflow: Autonomous Research Loop

Este loop permite que o Especialista refine uma pesquisa de forma independente até que ela atinja os padrões de qualidade e "interesse" da Melanora.

## 📋 Passos da Iteração

1.  **Exploração Inicial**: Realizar uma busca ampla usando o [Search Jina](https://s.jina.ai/).
2.  **Avaliação de Interesse**: Aplicar os [Critérios de Interesse](file:///c:/Users/Newton/Meu%20Drive/1.%20Projetos/Melanora/02_Oficios_Especialidades/Especialista_Pesquisa/interesting_research_criteria.md).
    - *Se a pesquisa for superficial:* Identificar "gaps" de informação e formular novas queries mais profundas.
3.  **Extração Segmentada**: Se o tópico for denso, extrair dados em camadas (Ex: Core Técnico -> Casos de Uso -> Críticas).
4.  **Auto-Correção**: Se encontrar informações contraditórias, o Especialista deve buscar uma terceira fonte para desempate (Triangulação).
5.  **Finalização**: Só interromper o loop quando houver pelo menos uma "Faísca Criativa" clara para o projeto.

*A autonomia nasce da persistência guiada por propósito.* 🚀
