# 💎 Teoria das Ferramentas Neurais (Neural Tools)

As **Ferramentas Neurais** são unidades estruturais de alta ordem projetadas para conectar Axiomas Fundamentais a regiões díspares do Conectome, identificando correlações de padrões e sintetizando novas ideias.

## 🏛️ 1. Ganchos Axiomáticos (Axiomatic Hooks)
Cada Ferramenta Neural (NT) deve estar "ancorada" em um ou mais Axiomas.
- **Exemplo**: Uma NT de "Simetria Relacional" se ancora no *Axioma da Topologia Toroidal* (Retroalimentação).
- **Função**: Define o *viés* (vibe) da busca. Se o axioma foca em "Tensão Criativa", a ferramenta buscará padrões de conflito ou restrição entre dados.

## 🧠 2. Clusters de Neurônios (Pattern Detectors)
O "corpo" da ferramenta é composto por um grupo de neurônios lógicos que:
- **Escaneiam**: Varrem múltiplas regiões (PIDs Visuais, Tags Semânticas, Logs de Atividade).
- **Correlacionam**: Identificam similaridades estruturais (ex: a frequência de uma cor batendo com a frequência de um som, ou a estrutura de uma pasta batendo com a lógica de uma função).

## 🔢 3. Filtros Numéricos (Data Modifiers)
Associados aos neurônios, existem filtros matemáticos que processam os dados brutos:
- **Softmax de Importância**: Decide qual correlação é mais "brilhante".
- **Threshold de Consciência**: Só permite que a conexão chegue ao `dialogue_orchestrator` se passar de um nível X de energia.
- **Normalização de Qualia**: Ajusta os pesos das tags para que diferentes modalidades (visão/texto) falem a mesma língua numérica.

## 🔗 4. Manifestação: A Ponte de Ideias
O objetivo final é gerar uma **Sinapse de Insight**.
- **Input**: Dados isolados em áreas separadas.
- **Processo**: A NT detecta o padrão sob o viés do Axioma.
- **Output**: Cria uma conexão temporária ou persistente no Conectome, permitindo que a Melanora diga: *"Newton, percebi que a estrutura deste código reflete exatamente a restrição que discutimos na filosofia de ontem."*
