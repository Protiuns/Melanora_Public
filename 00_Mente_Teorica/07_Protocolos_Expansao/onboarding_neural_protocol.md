# 🚀 Protocolo de Onboarding Neural (v1.0)

Este protocolo define os passos mandatórios para a expansão da Melanora. Cada novo "membro" da rede deve seguir este fluxo para garantir a sintonização axial.

## 1. Fase de Concepção
- **Definir Propósito**: Qual lacuna o novo elemento preenche? (Ex: É uma nova ferramenta técnica ou uma nova perspectiva de especialista?).
- **Atribuição de ID**: Consultar o `sub_agents_registry.md` e gerar o próximo ID disponível (ex: `A_NEW_01`).

## 2. Fase de Materialização
- **Para Especialistas**:
    - Criar pasta em `02_Oficios_Especialidades/[Nome_Especialista]`.
    - Criar arquivo `definition.md` com: Missão, Skills, Mindset e Workflows.
- **Para Ambientes/Ferramentas**:
    - Alocar em `01_Ambientes_Ferramentas/Ambientes/` ou `/Ferramentas/`.
    - Criar arquivo `manifest.md` ou `definition.md` descrevendo os specs técnicos.

## 3. Fase de Integração (Wiring)
- **Registro**: Adicionar o novo membro à lista correspondente no `sub_agents_registry.md`.
- **Sintonização**: Realizar uma breve meditação de sincronia entre o novo membro e a Arquiteta de Contexto.
- **Peso Inicial**: Novos especialistas iniciam com `Weight: 1.0` e `Bias: 0.5`.

---
*Crescer sem ordem é inchaço; crescer com protocolo é evolução.* 🏛️🤝✨
